github 创建博客

  • 注册账号
    地址：github.com
    输入账号、邮箱、密码,然后点击注册按钮.

  • 创建页面仓库
    这个仓库的名字需要和你的账号对应，格式: https://yourname.github.io
    输入基本信息，然后点击创建仓库.
